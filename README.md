A natural language chat assistant for WordPress by **Regionally Famous**.

## What is this?
Hey Wapuu replaces the clunky command palette with a friendly chat bubble. Ask Wapuu to do things in plain English, and he'll help you navigate your site!

## Production Installation
1. Download the `hey-wapuu.zip` file.
2. Go to your WordPress Admin > Plugins > Add New > Upload Plugin.
3. Upload and activate.

## Development
- `npm install`
- `npm run build` to compile assets.
- `npm start` to run with a watcher.
